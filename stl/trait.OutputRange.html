Expand description

Models a fully mutable range.

OutputRange is extension of SemiOutputRange.

There are 2 types of safe mutation possible for ranges:

  1. Reordering of elements that is provided by SemiOutputRange.
  2. Element Assignment: Assign different value to element at any position. For the same, at_mut function is required that returns mutable reference to element at any position.
