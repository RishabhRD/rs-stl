fn for_each<F>(&self, f: F)
where F: FnMut(&Self::Element),

Applies f to each element of collection.

§Complexity:
  • O(n) where n == self.count().
§Example
use stl::*;

let arr = [1, 2, 3];
let mut sum = 0;
arr.for_each(|x| sum = sum + x);
assert_eq!(sum, 6);

fn full(&self) -> Slice<'_, Self::Whole>

Returns slice of the collection covering full collection.

§Complexity

fn prefix(&self, max_length: usize) -> Slice<'_, Self::Whole>

Returns a slice, upto specified maximum length, containing the initial elements of collection.

§Postcondition
  • If the maximum length exceeds the number of elements in the collection, the result contains all elements in the collection.
§Complexity
  • O(1) for RandomAccessCollection; otherwise O(k) where k is the number of elements in resultant slice.
§Example
use stl::*;

let arr = [1, 2, 3, 4, 5];
let s = arr.prefix(3);
assert!(s.equals(&[1, 2, 3]));
Source

fn drop(&self, count: usize) -> Slice<'_, Self::Whole>

Returns a slice containing all but the given number of initial elements.

§Postcondition
  • If count > self.count(), returns an empty slice.
§Complexity
  • O(1) for RandomAccessCollection;
  • O(k) otherwise, where k is number of elements to drop from beginning of collection.
§Examples
use stl::*;

let arr = [1, 2, 3, 4, 5];
let s = arr.drop(3);
assert!(s.equals(&[4, 5]));

where OtherCollection: Collection, F: FnMut(&Self::Element, &OtherCollection::Element) -> bool,

Returns true if elements of self is equivalent to elements of other by given relation bi_pred.

§Postcondition
  • Returns true if elements of self is equivalent to other by given relation bi_pred.
  • If self and other have different number of elements, then return false.
§Complexity
  • O(min(m, n)) where

    • n == other.count()

fn partition_point<F>(&self, belongs_in_second_half: F) -> Self::Position
where F: FnMut(&Self::Element) -> bool,

Returns position of first element of collection for which predicate returns false.

§Precondition
  • The collection should be already partitioned wrt predicate i.e, there exist a position i such that predicate is true for every element of self.prefix_upto(i) and predicate is false for every element of self.suffix_from(i).
§Complexity
  • O(log n) for RandomAccessCollection, O(n) otherwise; where n == self.count().
  • O(log n) application of belongs_in_second_half.
§Example
use stl::*;

let arr = [1, 3, 5, 2, 4];
let i = arr.partition_point(|x| x % 2 == 0);
assert_eq!(i, 3);

fn fold_left<R, F>(&self, init: R, op: F) -> R
where F: FnMut(R, &Self::Element) -> R,

Returns the result of combining elements of given collection using given accumulation operation from left to right.

§Postcondition
  • Result is (((init + e1) + e2) + ... + en). where e1, e2, …, en are the references to collection elements.
§Complexity:
  • O(count)
§Examples
use stl::*;

let arr = [1, 2, 3];
assert_eq!(arr.fold_left(0, |x, y| x + y), 6);
