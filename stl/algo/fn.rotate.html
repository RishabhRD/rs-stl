Expand description

Rotates the given range at mid.

§Precondition

  • [start, mid) represent valid position in rng.
  • [mid, end) represent valid position in rng.

§Postcondition

  • Swaps element of [start, end) in such a way that the elements at [start, mid) are placed after elements at [mid, end) while the orders of both ranges are preserved.
  • Returns position to element originally at start.
  • Complexity: O(n). At most n swaps.

Where n is number of elements in [start, end).
