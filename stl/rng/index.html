§Ranges module

The rng module provides a collection of range version of STL algorithms as well as some infix version for some subset of algorithms.

If infix version of range algorithm is supported, then it is in following format:

use stl::*;
use rng::infix::*;

let arr = [1, 2, 3];

let cnt = rng::count_if(&arr, |x| x % 2 == 1);
assert_eq!(cnt, 2);
let cnt = arr.count_if(|x| x % 2 == 1);
assert_eq!(cnt, 2);

For all algorithms it is mentioned in doc if infix version is supported or not for that algorithm. Or open infix module to see all supported infix algorithms.

For using infix version of supported algorithms use:

use stl::rng::infix::*;
