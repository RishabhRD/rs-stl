Expand description

Models a multi-pass linear sequence of elements.

Representation:

  _ _ _ _ _ _

  ^            ^
  |            |
start   -->   end

Required Associated Types§

Source

type Position: Regular + Ord

Type of positions in the collection.

Source

type Element

Type of element in the collection.

Source

type ElementRef<'a>: Deref<Target = Self::Element> where Self: 'a

Type that is like &Element. For collections whose elements are in memory, its simply &Element.

Source

type Whole: Collection<Position = Self::Position, Element = Self::Element, Whole = Self::Whole>

Type representing whole collection. i.e., Self == Slice<W> ? W : Self

Provided Methods§

Source

fn form_next_n(&self, position: &mut Self::Position, n: usize)

Increments given position by n.

§Precondition
  • There are n valid positions in self after position.
§Complexity
  • O(1) for RandomAccessCollection; O(n) otherwise.
Source

fn form_next_n_limited_by( &self, position: &mut Self::Position, n: usize, limit: Self::Position, ) -> bool

Increments given position by n, or so that it equals the given limit.

§Precondition
  • limit is a valid position in self.
  • limit should be increment-reachable from position.
§Postcondition
  • Returns true if position has been incremented exactly by n without going beyond limit, otherwise returns false.
  • When the return value is false, the value position is equal to limit.
§Complexity
  • O(1) for RandomAccessCollection; O(n) otherwise.
Source

fn next(&self, position: Self::Position) -> Self::Position

Returns position immediately after position.

§Precondition
  • position != end()
Source

fn next_n(&self, position: Self::Position, n: usize) -> Self::Position

Returns nth position after position.

§Precondition
  • There are n valid positions in self after position.
§Complexity
  • O(1) for RandomAccessCollection; O(n) otherwise.
Source

fn next_n_limited_by( &self, position: Self::Position, n: usize, limit: Self::Position, ) -> Option<Self::Position>

Returns nth position after given position, unless n is beyond limit.

§Precondition
  • limit is a valid position in self.
  • limit should be increment-reachable from position.
§Postcondition
  • Returns nth position after given position, unless n is beyond limit.
  • Otherwise, returns None.
§Complexity
  • O(1) for RandomAccessCollection; O(n) otherwise.
Source

fn distance(&self, from: Self::Position, to: Self::Position) -> usize

Returns number of elements in [from, to).

§Precondition
  • [from, to) represents valid positions in the collection.
§Complexity
  • O(1) if RandomAccessCollection; O(n) otherwise.
Source

fn count(&self) -> usize

Returns number of elements in collection.

§Complexity
  • O(1) if RandomAccessCollection; O(n) otherwise.
Source

fn underestimated_count(&self) -> usize

Returns count less than or equal to number of elements in collection.

§Complexity
  • O(1) if RandomAccessCollection; O(n) otherwise.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
