pub fn split_evenly_in_with_min_size( self, max_slices: usize, min_size: usize, ) -> SplitEvenlyIterator<'a, Whole> โ

Splits self into at max max_slices slices with each slice being of at min size of min_size by returning an Iterator of slices.

ยงPrecondition
  • max_slices > 0,
ยงPostcondition
  • If splitting exactly evenly is not possible, then slices on start would have bigger size than slices at end, still maintaining as even splitting as possible.
ยงComplexity
  • O(1) for RandomAccessCollection; otherwise O(n) where n == self.count().
ยงExample

    .split_evenly_in_with_min_size(3, 2)
    .map(|s| s.iter().copied().collect())
    .collect();
assert_eq!(splits, vec![vec![1, 2, 3], vec![4, 5], vec![6, 7]]);
