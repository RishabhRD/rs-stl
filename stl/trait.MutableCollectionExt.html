fn for_each_mut<F>(&mut self, f: F)
where F: FnMut(&mut Self::Element),

Applies f to each element of collection.

§Complexity
  • O(n) where n == self.count().
§Example
use stl::*;

let mut arr = [1, 2, 3];
arr.for_each_mut(|e| *e = *e + 1);
assert_eq!(arr, [2, 3, 4]);
