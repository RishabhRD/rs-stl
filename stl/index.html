§STL module

The stl module provides formal definition of “Collections” in terms of traits. Based on primitives exposed by traits, the module provides many generic algorithms for types satisfying “Collection” traits.

The module also implements the “Collection” related traits for stdlib data structures. Currently “Collection” traits have been implemented for:

  • [T;n] (array)
  • [T] (slice)
  • Vec<T> (Vec)
  • Option<T> (Option)
  • Range<T> (a..b) where T is a signed/unsigned integer type.
  • RangeInclusive<T> (a..=b) where T is a signed/unsigned integer type.
